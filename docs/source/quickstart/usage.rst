Usage
=====

.. _installation:

Installation
------------

To use Infodeslib, first install it using pip:

.. code-block:: console

   $ pip install infodeslib


Mandatory Requirements
-----------------------------

You need to install following packages before using our library: 

* **shap**
* **matplotlib**
* **numpy**
* **sklearn**

Creating usage
----------------

For example:

>>> import infodesib
>>> import infodesib.des.KNORAE 
>>> infodesib.get_version()
